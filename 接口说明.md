好的，按功能分组，给出“完整地址 + 作用”（域名统一 https://dtwz.zhiyuansafety.com）：

公共接口（无需鉴权）
- 健康检测
  - GET https://dtwz.zhiyuansafety.com/api/health
  - 作用：检查后端服务健康状态
- FAQ 列表
  - GET https://dtwz.zhiyuansafety.com/api/faqs
  - 作用：获取前台展示的FAQ（仅is_active=true）
- 联系表单提交
  - POST https://dtwz.zhiyuansafety.com/api/contacts
  - 作用：提交“Get in Touch”表单
- 分类列表
  - GET https://dtwz.zhiyuansafety.com/api/categories
  - 作用：获取分类（含封面、产品数量等）用于首页“Safety Shoe Categories”
- 产品列表（带汇总信息）
  - GET https://dtwz.zhiyuansafety.com/api/products
  - 作用：获取产品列表（支持排序/分页参数，供前台或管理端列表用）
- 指定产品详情
  - GET https://dtwz.zhiyuansafety.com/api/products/:id
  - 作用：获取单个产品完整信息（含图片、详细文本）
- 分类下产品
  - GET https://dtwz.zhiyuansafety.com/api/categories/:slug/products
  - 作用：按分类slug获取该类产品，供首页分类卡片点击弹窗

认证（获取管理JWT）
- 管理登录
  - POST https://dtwz.zhiyuansafety.com/api/auth/login
  - 作用：管理员登录，返回JWT（admin_token）

管理接口（需请求头 Authorization: Bearer <admin_token>）
- 仪表盘统计
  - GET https://dtwz.zhiyuansafety.com/api/admin/dashboard
  - 作用：获取总产品、发布产品、分类数、联系表单统计、FAQ数
- FAQ 管理
  - GET https://dtwz.zhiyuansafety.com/api/faqs/admin
    - 作用：获取全部FAQ（含隐藏项）
  - POST https://dtwz.zhiyuansafety.com/api/faqs/admin
    - 作用：新增FAQ
  - PUT https://dtwz.zhiyuansafety.com/api/faqs/admin/:id
    - 作用：编辑FAQ
  - DELETE https://dtwz.zhiyuansafety.com/api/faqs/admin/:id
    - 作用：删除FAQ
- 联系表单管理
  - GET https://dtwz.zhiyuansafety.com/api/contacts/admin
    - 作用：查询提交列表（支持status筛选、分页）
  - GET https://dtwz.zhiyuansafety.com/api/contacts/admin/:id
    - 作用：查看单条详情
  - PUT https://dtwz.zhiyuansafety.com/api/contacts/admin/:id
    - 作用：更新状态/备注
  - DELETE https://dtwz.zhiyuansafety.com/api/contacts/admin/:id
    - 作用：删除提交
  - GET https://dtwz.zhiyuansafety.com/api/contacts/admin/stats/summary
    - 作用：联系表单统计汇总
- 产品管理
  - GET https://dtwz.zhiyuansafety.com/api/admin/products
    - 作用：产品列表（管理视图）
  - POST https://dtwz.zhiyuansafety.com/api/admin/products
    - 作用：新增产品（含图片URL数组、结构化文本）
  - PUT https://dtwz.zhiyuansafety.com/api/admin/products/:id
    - 作用：编辑产品（全量或主要字段）
  - DELETE https://dtwz.zhiyuansafety.com/api/admin/products/:id
    - 作用：删除产品
- 分类管理（如启用）
  - GET https://dtwz.zhiyuansafety.com/api/admin/categories
  - POST https://dtwz.zhiyuansafety.com/api/admin/categories
  - PUT https://dtwz.zhiyuansafety.com/api/admin/categories/:id
  - DELETE https://dtwz.zhiyuansafety.com/api/admin/categories/:id
  - 作用：后台维护分类及排序、启用状态、关联封面产品等
- 图片上传（R2，后端转存）
  - POST https://dtwz.zhiyuansafety.com/api/upload/image
  - 作用：上传图片至R2并返回URL（body: multipart/form-data，字段名 image）

提示
- 前台已使用同域 https://dtwz.zhiyuansafety.com，并统一以 /api 为前缀，后端部署后即可直接连通。
- 管理端页面的所有 /admin 路由均需携带 JWT；公共路由无需。
