# 登泰劳保鞋网站 - 启动指南

首页显示：现在已经正常从 API 获取数据并显示
性能策略：采用了 SSG (静态生成) 技术，保证了加载速度最快，同时又能通过后台管理内容。这是目前最先进的独立站架构方案。

## 🚀 快速启动

### 1、启动数据库（Docker）
```bash
cd "/Users/zhengpeilong/1-正式项目/2-登泰网站"
docker-compose up -d
```

### 2、启动网站后台（Strapi）
```bash
cd "/Users/zhengpeilong/1-正式项目/2-登泰网站/safetyshoe-backend"
npm run develop
```
访问地址：http://localhost:1337/admin

### 3、启动网站前端（Next.js）
```bash
cd "/Users/zhengpeilong/1-正式项目/2-登泰网站/safetyshoe-frontend"
npm run dev
```
访问地址：http://localhost:3000 或 http://localhost:3001

## 📋 服务说明

| 服务 | 端口 | 说明 |
|------|------|------|
| PostgreSQL | 5434 | 数据库（Docker） |
| Strapi 后台 | 1337 | 产品/新闻管理 |
| Next.js 网站 | 3000/3001 | 对外展示网站 |

---

## 🛑 停止服务

### 停止数据库
```bash
docker-compose down
```

### 停止后台和前端
按 Ctrl+C 停止终端中运行的服务

## ✅ 核心功能

1. **产品管理**：在 Strapi 后台添加/编辑产品
2. **新闻管理**：发布工厂动态和新闻

4. **媒体库**：统一管理所有图片

## 🆘 遇到问题？

- **端口被占用**：修改 docker-compose.yml 中的端口
- **权限错误**：检查 Strapi 后台的 Public 角色权限
- **产品不显示**：确保已发布产品并设置了 API 权限
