# 登泰安全鞋网站 - 产品后台录入指南

填报示例 (Example)

假设我们要录入一款 **S3 标准的高帮安全鞋**：

1.  **Name**: Heavy Duty Work Boot
2.  **Model Code**: SL-905
3.  **Safety Standard**: 选择 `S3`
4.  **Style**: 选择 `High Boot`
5.  **Industries**: 输入 `["Construction", "Mining"]`
6.  **Features**: 输入 `["Waterproof", "Anti-static", "Steel Toe"]`
7.  **Materials**:
    *   **Upper**: Crazy Horse Leather
    *   **Outsole**: Rubber
    *   **Toe Cap**: Composite
8.  **Is Hot**: `True` (开启)


## 一、 基础信息 (Basic Info)

这些是产品的核心信息，必须准确填写。

| **Name** | 产品名称  
**1. 列表页**：显示在产品卡片下方。<br>
**2. 弹窗页**：显示在最顶部的醒目标题。<br>
**3. 搜索**：用户搜索时的主要匹配字段。 |


| **Model Code** | 产品型号 
 **1. 列表页**：显示在产品名称下方 (如 `Model: SL-001`)。<br>
 **2. 弹窗页**：显示在标题下方的副标题。<br>
 **3. 搜索**：用户可以通过搜索型号找到产品。 |

| **Images** | 产品图片 
**1. 第一张图**：作为列表页的封面图 (Cover)。<br>
**2. 所有图片**：在弹窗页左侧形成**图片轮播/画廊**，用户可以点击切换查看大图。 |


| **Description** | 产品描述 | Long Text | 否 | **弹窗页**：在"询盘表单"上方或单独的描述区域显示，用于详细介绍产品卖点。 |



| **Price Range** | 价格范围 | Text | 否 | **不直接显示** (目前前端暂未展示价格，仅作为询盘参考或未来扩展使用)。 |



## 二、 分类与筛选 (Classification & Filters)

这些字段直接决定了用户在**产品列表页左侧侧边栏**筛选时，能否找到该产品。请务必勾选准确。

| **Safety Standard** | 安全标准 | Dropdown | **核心筛选**。<br>
**1. 筛选**：对应左侧 `Standard` 过滤器 (SB, S1, S1P, S2, S3)。<br>
**2. 显示**：在列表卡片和弹窗标题上方显示为一个小标签 (Badge)。 |

| **Style** | 鞋款风格 | Dropdown | 
**1. 筛选**：对应左侧 `Style` 过滤器 (Low Cut, High Cut, Sandal 等)。<br>
**2. 显示**：在弹窗标题上方显示为灰色标签。 |

| **Industries** | 适用行业 | JSON (多选) | 
**1. 筛选**：对应左侧 `Industry` 过滤器 (Construction, Mining, Logistics 等)。<br>
**2. 显示**：在弹窗标题上方显示为淡蓝色标签。 |

| **Features** | 功能特性 | JSON (多选) | 
**1. 筛选**：对应左侧 `Feature` 过滤器 (Waterproof, Anti-static, Slip Resistant 等)。<br>
**2. 显示**：在弹窗页中部，以图标+文字的小标签形式展示。 |


| **Additional Certs** | 额外认证 | JSON (多选) | **显示**：在弹窗标题上方显示为边框标签 (如 ASTM, CE)。不参与主要筛选。 |

---

## 三、 详细规格 (Detailed Specifications)

这些字段用于在弹窗详情页中展示产品的技术参数，通常显示在一个灰色的 **"Material Specs" (材料规格)** 区域内。

| **Materials** | 材料组件 | Component | 点击展开后填写以下子字段： |
| - **Upper** | 鞋面材料 | Text | 显示在规格列表的 `Upper` 一栏 (如: Genuine Leather)。 |
| - **Outsole** | 大底材料 | Text | 显示在规格列表的 `Outsole` 一栏 (如: PU/PU Dual Density)。 |
| - **Toe Cap** | 鞋头材料 | Text | 显示在规格列表的 `Toe Cap` 一栏 (如: Steel Toe)。 |
| - **Midsole** | 中底材料 | Text | 显示在规格列表的 `Midsole` 一栏 (如: Steel Plate)。 |
| - **Lining** | 内里材料 | Text | 显示在规格列表的 `Lining` 一栏 (如: Breathable Mesh)。 |

---

## 四、 营销标记 (Marketing Flags)


| **Is Hot** | 热门产品 | Boolean (开关) | **列表页**：如果开启，产品卡片右上角会显示红色的 "HOT" 标签。 |

| **Is New** | 新品 | Boolean (开关) | **列表页**：如果开启，产品卡片右上角会显示绿色的 "NEW" 标签。 |

| **MOQ** | 起订量 | Text | **弹窗页**：显示在底部的 "MOQ" 区域 (默认显示 500 Pairs)。 |

---


## 五、 多语言注意事项

*   Strapi 后台右侧有 **Locales** 切换 (English / Chinese)。
*   **必须分别填写**：
    *   `Name` (英文名 / 中文名)
    *   `Description` (英文描述 / 中文描述)
    *   `Materials` 里的各个子字段 (英文材料名 / 中文材料名)
*   **无需重复填写** (自动同步)：
    *   `Images` (图片是通用的)
    *   `Model Code` (型号通用)
    *   `Safety Standard` (标准代号通用)
